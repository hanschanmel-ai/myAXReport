SvelteKit frontend scaffolding

Setup

1. npm create svelte@latest frontend-svelte
2. cd frontend-svelte
3. npm install
4. npm run dev

Pages

- Suite-style top menu with dropdowns
- Inventory: item tabs, movements, transfer, adjustment, locations, bins
- Transactions: PO/SO create and receive/ship
- Accounting: GL post and list
- Lists: vendors, customers, items
- Support: AI assistant
